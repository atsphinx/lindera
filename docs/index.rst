================
atsphinx-lindera
================

概要
====

Linderaを利用したSphinx向け機能を提供するための拡張ライブラリです。

LinderaはRustで実装された形態素解析ライブラリです。
詳細は下記のURLを参照してください。

* https://github.com/lindera/lindera
* https://lindera.github.io/lindera/

現在、Sphinxドキュメントの全文検索のパフォーマンス向上に利用できます。
